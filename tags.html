Important HTML

This page is designed as a quick reference for some common tags covered in Learn Enough HTML to Be Dangerous. In the process of making, we'll learn how to make HTML tables via tableand related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings including a heading (levels 1–6)
p paragraph include paragraph of text
table table include table
tr table row include a row of data
th table header include a table data cell
td table data include a table data cell

Inline Elements

Tag Name Purpose Example Results
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make strong text <strong>at least a billion</strong> at least a billion
a Anchor make hyperlink <a href="http://learnenough.com/">Learn Enough</a> Learn Enough
img Image Include an Image <img src=Kitten.jpg>
code Code Designed to display pieces of markup or source code <code> Technical Sophistication</code> Technical Sophistication
